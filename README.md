# Verilog Basics

This repository contains Verilog HDL implementations of fundamental digital circuits.

## Topics Covered
- Logic Gates
- Multiplexers (MUX)
- Decoders
- Encoders
- Comparators
- Adders and Subtractors

## Tools Used
- Verilog HDL
- Xilinx Vivado
- GitHub

## Goal
Build a strong foundation in RTL Design and FPGA development.
